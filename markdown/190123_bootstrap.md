## Bootstrap

> 반응형, 모바일용 자바 스크립트 툴

* 편하게 작업하기
  * open in browser, htmltagwrap 확장 프로그램 설치
  * alt + b : html 실행
  * alt + w : 태그 생성
  * alt + shift + f : 코드 정리
  * alt+ctrl : 멀티 커서

* 작업 순서

  * reboot (일반적으로 css reset) 코드 복사

  * bootstrap 홈페이지에서 css style sheet 복사 (title 바로 아래)

    

  * js 코드(스크립트 태그) body 안에 복사

* 부트 스트랩은 모든 작업을 '클래스'로 한다.

* 작업 참고사항

  * 컨텐츠 넣을 공간을 class container로 묶고 시작

* 그리드(grid) 시스템
